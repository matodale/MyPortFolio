Hello, I'm Matodale

저는 웹 개발자에 도전하는 개발자입니다.

처음 시작은 학원에서 일러스트와 포토샵을 배우는 것부터 시작하였습니다. 주로 서적을 이용하는 공부 방법으로 기본에 충실하고자 노력하였습니다.

Java 웹 프로그래밍에 관심이 많아서 JSP page를 만들다가 유튜브 영상들을 접하면서 Spring Boot배우게 되었습니다. 현재는 전자정보프레임워크를 이용한 개발을 준비하고 있습니다.

프론트엔드에 대해 기본 부터 시작하였지만 JavaScript 에 대해 어려워 기본 부터 다시 배우고자 노력하여 Java 기본부터 다시 최근에 시작하였습니다.
